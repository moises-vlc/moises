# PREPARANDO EL DOCKER FILE Y CREANDO LA IMAGEN

**Creamos una carpeta para la práctica**
```bash
sudo mkdir ubutunano
cd ubuntunano
```

**Crearemos la imagen con:**
```bash
sudo docker build -t ubuntunano ./
```

**Crearemos el Dockerfile**
```bash
sudo nano Dockerfile
```
