# DECARGARMOS LA APLICACIÓN
Nos descargamos la aplicación:
```bash
git clone https://github.com/docker/getting-started.git app-source
cp -r app-source/app/* ./
```

# PREPARAMOS EL DOCKERFILE
Creamos y entramos a la carpeta de la práctica
```bash
mkdir Caso4-2
cd Caso4-2
```
Creamos el fichero Dockerfile
```bash
sudo nano Dockerfile
```
Y pegaremos esto y guardamos
<img width="807" height="524" alt="image" src="https://github.com/user-attachments/assets/b26459a1-f227-44a8-9c43-13812dc4997e" />

# Construir la imagen
Desde nuestra carpeta donde está el Dockerfile:
```bash
docker build -t sampledocker ./
```
<img width="802" height="350" alt="image" src="https://github.com/user-attachments/assets/7d9a4e13-d835-429b-a43c-87ebf36cb69b" />

# Ejecutar el contenedor y exponer el puerto 
Ejecutamos:
```bash
docker run -dp 3000:3000 --name mi_app_sample sampledocker
```
# Probar la aplicación
Abrimos un navegador y ponemos:
```bash
http://localhost:3000
```
