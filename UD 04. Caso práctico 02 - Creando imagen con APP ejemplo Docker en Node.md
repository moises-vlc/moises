# DECARGARMOS LA APLICACIÓN
Nos descargamos la aplicación:
```bash
git clone https://github.com/docker/getting-started.git app-source
cp -r app-source/app/* ./
```

# PREPARAMOS EL DOCKERFILE
Creamos y entramos a la carpeta de la práctica
```bash
mkdir Caso4-2
cd Caso4-2
```
Creamos el fichero Dockerfile
```bash
sudo nano Dockerfile
```
