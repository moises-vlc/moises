# Practica de comandos en contendor Docker.pdf".
## Preparamos el docker
<p>
Instalamos docker primero porque en nuestra máquina Ubuntu no lo tenemos instalado
  
```bash
sudo snap install docker
```
Creamos nuestro docker

```bash
docker run -it --name ejercicio ubuntu /bin/bash
```
Para arrancar nuestro docker

```bash
docker start -ai ejercicio
```
Una vez dentro creamos 10 carpetas manualmente con:

```bash
mkdir /root/01
mkdir /root/02
mkdir /root/03
mkdir /root/04
mkdir /root/05
mkdir /root/06
mkdir /root/07
mkdir /root/08
mkdir /root/09
mkdir /root/10
```
<img width="374" height="225" alt="image" src="https://github.com/user-attachments/assets/876ea26b-f772-4602-b8fb-523224f1e7e0" />

Nos metemos dentro de root en el contenedor y hacemos un ls para ver que estan todas las carpetas

```bash
sudo docker start -ai ejercicio
cd root
ls
```
<img width="492" height="113" alt="image" src="https://github.com/user-attachments/assets/4cad9188-8716-42aa-bdb5-eebb19096a34" />

Salimos del contenedor

```bash
exit
```

Iniciamos el contenedor en segundo plano

```bash
sudo docker start ejercicio
```

Creamos el script

```bash
sudo nano comprobar_practica.sh
```

Y pegamos todas estas líneas
<img width="811" height="731" alt="image" src="https://github.com/user-attachments/assets/6338104b-6c0b-4119-84e4-ca6dedd51208" />

Darle permisos al archivo

```bash
sudo chmod +x comprobar_practica.sh
```

Y definitivamente ya ejecutamos el script

```bash
sudo ./comprobar_practica.sh ejercicio
```
