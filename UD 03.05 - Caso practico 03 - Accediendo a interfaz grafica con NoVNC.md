# Preparamos el contenedor 
Nos vamos a la terminal de Linux y ejecutamos

```bash
docker run --rm -it -p 8080:8080 theasp/novnc
```
Se ejecutará un docker temporal

# Accediendo a la interfaz gráfica
Abrimos un navegador y ponemos esta URL para acceder a noVNC

```bash
 http://localhost:8080/vnc.html
```
